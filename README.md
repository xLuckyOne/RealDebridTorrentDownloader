Automatically checks your clipboard and adds all magnet links to Read Debrid and downloads them.


Change following variables:


| Variable        | Content           |
| ------------- |:-------------:| 
| secretKey      | your Secretkey, obtainable via https://real-debrid.com/apitoken |


| Required Libraries|
| -------------| 
| win32clipboard|
| time|
| threading|
| requests|
| tqdm|
